Amazon Payments plugin allows you to create payment buttons to accept payment for products or services via Amazon.

== Description ==

This plugin allows you to create payment buttons that you can use to accept payment via Amazon payment gateway. You will need to have an Amazon merchant account.

View plugin documentation on the [amazon payments plugin page](https://www.tipsandtricks-hq.com/amazon-payments-plugin-for-wordpress).

= How to use the plugin =

1. Install and activate the plugin.
2. Go to the plugin settings page ( on dashboard 'Settings' -> 'Amazon Payments' ) and edit the necessary options.

=  How do I use a shortcode to create a payment button =

1. If you would like to add the button pay to your website, just copy and paste this shortcode to your post or page or widget, and select Options: [amazon_payments amount='' description='' 	currency='']
2. Description Pay button - [amazon_payments amount="" description="" currency=""]
	Amount - how much will it cost
	Description - a product name
	Currency - the currency in which this product will be sold

= How to get data for the plugin =

1. Here is a link to create Amazon Sandbox - https://payments.amazon.com/help/81755
2. Here is a link to create a Free Account Amazon - http://aws.amazon.com/

== Installation ==

1. Upload `amazon-payments` folder to the `/wp-content/plugins/` directory
2. Activate the plugin via the 'Plugins' menu in WordPress.
3. Plugin settings are located in 'Settings', 'Amazon Payments'.
